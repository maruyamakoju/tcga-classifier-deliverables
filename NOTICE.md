# Notice

This repository contains code, documentation, trained lightweight model exports,
and compact validation artifacts for a research RNA-seq classifier.

The model and validation artifacts were derived from public cancer genomics
resources including TCGA, CPTAC, GTEx, GDC, and UCSC Xena datasets. Source data
remain subject to their original provider terms, access rules, citation
requirements, and data-use policies. The MIT license in this repository covers
the repository code and project-authored documentation; it does not override
third-party data provider terms.

This project is for research use only. It is not a clinical diagnostic model
and must not be used for patient management.
